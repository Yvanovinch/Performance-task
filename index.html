Performance Task - Loops and Iteration

1). Construct if and if... else statements that meet the following:

> Display a dialog box confirming a user is 25 years old and up.

> Display a dialog box confirming a user entered a number less than 100.

2). Construct for loops that meet the following tasks:

> Print the numbers 0-30, one number per line.

> Print only the even values from 0-40, one number per line.

> Print the numbers 40 to 10 in descending order, but only if the numbers are multiples of 3.
